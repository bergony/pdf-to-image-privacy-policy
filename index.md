Política de Privacidade — PDF to Image Converter

Última atualização: 2026-01-15

O aplicativo PDF to Image Converter respeita a privacidade dos usuários e foi desenvolvido para funcionar de forma simples, segura e transparente.

Coleta de informações

O PDF to Image Converter não coleta, armazena ou compartilha dados pessoais dos usuários.
O aplicativo funciona totalmente offline e não exige cadastro, login ou fornecimento de informações pessoais.

Uso de arquivos

O aplicativo acessa arquivos PDF armazenados no dispositivo somente quando o usuário seleciona explicitamente um arquivo para conversão.
As imagens geradas são salvas localmente no dispositivo do usuário e não são transmitidas para servidores externos.

Compartilhamento de dados

Nenhuma informação é compartilhada com terceiros.
O aplicativo não utiliza serviços de análise, rastreamento ou publicidade nesta versão.

Segurança

Como não há coleta ou armazenamento de dados pessoais, o risco de exposição de informações é minimizado. Todo o processamento ocorre localmente no dispositivo do usuário.

Permissões

As permissões solicitadas pelo aplicativo são utilizadas exclusivamente para:

Selecionar arquivos PDF no dispositivo

Salvar imagens convertidas no armazenamento local

Essas permissões são necessárias para o funcionamento básico do aplicativo.

Crianças

O aplicativo não é direcionado a crianças e não contém conteúdo inadequado. Ele pode ser utilizado por usuários com idade a partir de 13 anos.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada no futuro para refletir melhorias ou alterações no aplicativo. Qualquer modificação será publicada nesta página.
